# طريقة استخدام نظام إدارة الجامعة المصغر

##  تشغيل التطبيق

### الطريقة الأولى: باستخدام ملف JAR
```bash
java -jar UniversitySystem.jar
```

### الطريقة الثانية: من الكود المصدري
```bash
# ترجمة الكود
javac -cp src src/com/university/*.java

# تشغيل التطبيق
java -cp src com.university.UniversityApp
```

---

##  واجهة التطبيق

تظهر واجهة رسومية مقسمة إلى 3 أقسام:

### 1. قسم إدارة بيانات الطلاب (الجانب الأيسر)
- **رقم الطالب (ID):** أدخل رقم الطالب الجامعي
- **اسم الطالب:** أدخل اسم الطالب الكامل
- **التخصص:** أدخل تخصص الطالب

### 2. قسم عرض البيانات (المنتصف)
- يعرض قائمة جميع الطلاب المسجلين
- يتم تحديثه تلقائياً بعد كل عملية

### 3. قسم معلومات الفريق (الأسفل)
- يعرض أسماء الطلاب المطورين للمشروع

---

##  العمليات المتاحة

### 1. **إضافة طالب جديد (Insert)**
1. أدخل رقم الطالب في حقل ID
2. أدخل اسم الطالب
3. أدخل التخصص
4. اضغط على زر "إضافة طالب (Insert)"
5. تظهر رسالة نجاح ويتم تحديث القائمة

**ملاحظة:** لا يمكن إضافة طالب بنفس ID موجود مسبقاً

### 2. **البحث عن طالب (Search)**
1. أدخل رقم الطالب في حقل ID
2. اضغط على زر "بحث عن طالب (Search)"
3. تظهر نافذة ببيانات الطالب إذا وجد

### 3. **تعديل تخصص الطالب (Update)**
1. أدخل رقم الطالب في حقل ID
2. أدخل التخصص الجديد في حقل التخصص
3. اضغط على زر "تعديل التخصص (Update)"
4. يتم تحديث بيانات الطالب والقائمة

### 4. **حذف طالب (Delete)**
1. أدخل رقم الطالب في حقل ID
2. اضغط على زر "حذف طالب (Delete)"
3. تظهر رسالة تأكيد ويتم حذف الطالب من القائمة

##  رسائل الخطأ الشائعة

### **خطأ في الإدخال:**
- "الرجاء إدخال جميع البيانات (الاسم والتخصص)" - عند ترك حقول فارغة
- "رقم الطالب يجب أن يكون رقماً صحيحاً" - عند إدخال نص بدلاً من رقم

### **خطأ في الإضافة:**
- "الطالب برقم X موجود بالفعل" - عند محاولة إضافة طالب مكرر

### **خطأ في البحث/التعديل/الحذف:**
- "لم يتم العثور على طالب برقم X" - عند البحث عن رقم غير موجود
- "فشل الحذف: لم يتم العثور على طالب برقم X" - عند محاولة حذف طالب غير موجود
